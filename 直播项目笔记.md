# anlian-主项目

1. composer update【依赖更新】
2.  composer fund【依赖寻找】
3. composer create-project topthink/think tp【创建tp】
4.  php think run【运行】

# anlian-派框架

运行

```
php -S localhost:8001
```

配置redis

```php
// anlian/PhalApi/Config/app.php

    'REDIS_HOST' => "localhost",// 服务
    'REDIS_AUTH' => "123456", // 密码
    'REDIS_PORT' => "6379",// 端口

```
